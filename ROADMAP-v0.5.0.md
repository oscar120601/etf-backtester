## 🔧 詳細功能規格

### 1. 滾動報酬分析 (Rolling Returns)

**功能描述**:
- 計算指定滾動期間（1年/3年/5年/10年）的年化報酬
- 顯示滾動報酬的統計分布
- 比較不同投資組合的滾動報酬表現

**API 設計**:
```python
POST /api/v1/backtest/rolling-returns
{
  "portfolio": [{"symbol": "VTI", "weight": 0.6}, {"symbol": "BND", "weight": 0.4}],
  "window_years": [1, 3, 5, 10],
  "start_date": "2010-01-01",
  "end_date": "2024-12-31"
}
```

**輸出指標**:
- 各期間的平均滾動報酬
- 標準差
- 最佳/最差滾動期間
- 正報酬期間比例

---

### 2. 相關性矩陣 (Correlation Matrix)

**功能描述**:
- 計算投資組合中各 ETF 之間的價格相關係數
- 視覺化熱力圖顯示
- 幫助理解分散化效果

**API 設計**:
```python
GET /api/v1/analysis/correlation-matrix?symbols=VTI,VOO,BND&period=3y
```

**相關性解讀**:
| 相關係數 | 關聯強度 | 顏色 |
|---------|---------|------|
| 0.8 ~ 1.0 | 極強正相關 | 深紅 |
| 0.5 ~ 0.8 | 強正相關 | 紅色 |
| 0.2 ~ 0.5 | 弱正相關 | 淺紅 |
| -0.2 ~ 0.2 | 無相關 | 白色 |
| -0.5 ~ -0.2 | 弱負相關 | 淺藍 |
| -0.8 ~ -0.5 | 強負相關 | 藍色 |
| -1.0 ~ -0.8 | 極強負相關 | 深藍 |

---

### 3. 壓力測試 (Stress Test)

**功能描述**:
- 模擬投資組合在歷史危機期間的表現
- 比較與基準（如 S&P 500）的相對表現
- 識別投資組合的脆弱性

**預設危機情境**:

| 情境名稱 | 時間區間 | 描述 |
|---------|---------|------|
| 2008 金融危機 | 2007-10 ~ 2009-03 | 次貸危機、雷曼倒閉 |
| 2020 疫情崩盤 | 2020-02 ~ 2020-03 | COVID-19 爆發 |
| 2022 通膨緊縮 | 2022-01 ~ 2022-10 | 聯準會激進升息 |
| 2022 債券熊市 | 2022 全年 | 債券價格大幅下跌 |
| 自訂情境 | 用戶選擇 | 指定日期區間 |

**輸出指標**:
- 危機期間總報酬
- 最大回撤
- 與基準比較的相對報酬
- 恢復時間

---

### 4. 投資組合優化器 (Portfolio Optimizer)

**功能描述**:
- 基於現代投資組合理論 (MPT) 計算效率前緣
- 給定風險承受能力推薦最佳配置
- 支援權重限制（如單一標的不超過 40%）

**優化目標選項**:
1. **最大夏普比率** (Max Sharpe Ratio) - 推薦
2. **最小風險** (Min Volatility)
3. **目標報酬** (Target Return)
4. **目標風險** (Target Risk)

**API 設計**:
```python
POST /api/v1/optimizer/mpt
{
  "symbols": ["VTI", "VOO", "BND", "VXUS"],
  "objective": "max_sharpe",
  "constraints": {
    "min_weight": 0.05,
    "max_weight": 0.40
  },
  "risk_free_rate": 0.045
}
```

**輸出**:
- 效率前緣曲線數據
- 推薦組合權重
- 預期報酬與風險
- 與目前組合比較

---

### 5. 通膨調整報酬 (Inflation-Adjusted Returns)

**功能描述**:
- 使用美國 CPI 數據調整名義報酬
- 顯示實質購買力變化
- 計算實質報酬率

**API 設計**:
```python
POST /api/v1/backtest/real-returns
{
  "portfolio": [...],
  "adjust_for_inflation": true,
  "start_date": "2010-01-01",
  "end_date": "2024-12-31"
}
```
