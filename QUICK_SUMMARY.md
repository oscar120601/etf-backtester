## 📈 核心功能一覽

### 回測分析
- **歷史回測**：自訂日期區間、再平衡策略、定期定額
- **蒙地卡羅模擬**：10-50 年預測、100-5000 次模擬
- **多組合比較**：最多 3 組並排比較

### 投資組合工具
- **MPT 優化器**：效率前緣、最大夏普比率、最小波動率
- **預設模板**：60/40、全天候、科技股等一鍵套用
- **存檔功能**：儲存多個投資組合配置

### 風險分析
- **壓力測試**：6 種歷史危機情境
- **回撤分析**：最大回撤、持續時間、恢復統計
- **滾動報酬**：1/3/5/10 年期間分析
- **相關性矩陣**：ETF 間相關性熱力圖

### 報告與匯出
- **PDF 報告**：完整分析報告書
- **CSV**：時間序列數據
- **Excel**：多工作表詳細分析

## 🚀 快速啟動

### 最簡單方式（Windows）
```bash
start-dev.bat
```

### 手動啟動
```bash
# 後端
cd backend
uvicorn app.main:app --reload --port 8000

# 前端（新終端）
cd frontend
npm run dev
```

## 💡 使用建議

### 新手入門
1. 使用「預設模板」快速開始
2. 執行回測查看基本績效
3. 嘗試「多組合比較」不同策略

### 進階分析
1. 使用「投資組合優化」找到最佳配置
2. 執行「壓力測試」評估風險承受能力
3. 查看「滾動報酬」了解長期表現穩定性
