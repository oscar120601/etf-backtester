## 4. 風險管理

| 風險 | 影響 | 機率 | 應對策略 |
|------|------|------|---------|
| 資料來源不穩定 | 高 | 中 | 多來源備援機制 |
| 計算效能不足 | 中 | 中 | 快取、平行計算 |
| 時程延誤 | 中 | 高 | 優先實作 MVP 功能 |
| 第三方 API 限制 | 中 | 低 | 速率限制、錯誤重試 |
| 資安漏洞 | 高 | 低 | 定期掃描、程式碼審查 |
